Install for Mac 10 seconds

  1. Press ⌘ Space, type Terminal, press Return
  2. Paste this line, press Return, and Missting opens:
curl -fsSL https://raw.githubusercontent.com/HaoSophareth/missting/main/install.sh | bash

Free beta, not yet notarized by Apple, so a normal download shows a false "damaged" warning. This installs the latest version and it updates itself from there.
